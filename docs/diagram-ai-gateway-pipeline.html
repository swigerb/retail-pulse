Inbound Policies REQUEST
1
set-backend-service
Routes request to Azure AI Foundry backend URL
2
authentication-managed-identity
Replaces client api-key with APIM's managed identity token for backend auth
3
azure-openai-token-limit
Enforces 10,000 tokens-per-minute rate limit per subscription. Returns 429 if exceeded.
4
azure-openai-emit-token-metric
Emits prompt/completion token counts to Application Insights customMetrics
Backend FORWARD
5
Circuit Breaker
Configured on backend entity. Trips on 429 responses, opens circuit for 1 minute to prevent cascading failures.
6
Forward to Azure AI Foundry

API-Level Diagnostics RESPONSE
7
applicationinsights (Layer 2)
API-level diagnostic sends token metrics to App Insights. Sampling: 100%, verbosity: information
8
azuremonitor + largeLanguageModel (Layer 3)
Parses LLM request/response bodies. Extracts token counts, model name, prompt/completion content. Populates ApiManagementGatewayLlmLog table.
Instance-Level Diagnostic Settings ALWAYS
9
GatewayLogs + GatewayLlmLogs (Layer 1)
Instance-level diagnostic settings send all gateway and LLM logs to Log Analytics workspace. GatewayLlmLogs category must be explicitly enabled.
